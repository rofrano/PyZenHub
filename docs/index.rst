Welcome to PyZenHub's documentation!
====================================

PyZenHub is a Python wrapper for the ZenHub REST API. It allows you to communicate
with ZenHub in a more pythonic way.

Requirements:

This package requires ```PyGithub`` to complete the capabilities that are missing
from the ZenHub REST API.
